*****************************************************************************************************************************

Assuming the zip file has been downloaded on the lab computer.

1. Extract the zip file, 6 files will be extracted,
	(i) main.cpp
	(ii) matrix_mul.cl
	(iii) Makefile
	(iv) Assignment3_CSE520_Report
	(v) README
	(vi) run.sh

2. open the terminal and go to the directory in which the files are extracted, run the command given below tomake the script executable.
	chmod +x run.sh

3. Then, run the make command on the terminal to compile the program, after compilation matrix_mul executable will be created.

4. After compilation run the command given below, to obtain the output. Press Enter after every result to ontain the next result.
	./run.sh
